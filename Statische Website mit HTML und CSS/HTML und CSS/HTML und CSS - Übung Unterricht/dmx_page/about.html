Über DMX

DMX, geboren als Earl Simmons, war ein einflussreicher amerikanischer Rapper und Schauspieler. Er war bekannt für seine aggressive Rap-Technik und seine kraftvollen Texte, die oft persönliche und soziale Themen behandelten.
