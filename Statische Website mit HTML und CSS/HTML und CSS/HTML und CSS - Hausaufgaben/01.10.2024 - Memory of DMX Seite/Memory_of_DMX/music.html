Weltweit hat DMX mehr als 74 Millionen Tonträger verkauft und seine Alben wurden in den Vereinigten Staaten insgesamt 14 Mal mit Platin ausgezeichnet. Außerdem wurde er dreimal für einen Grammy Award nominiert.

Seine erfolgreichsten Songs sind:

  • Party Up aus dem Jahr 1999
  • Where the Hood At?
  • X Gon' Give It To Ya
