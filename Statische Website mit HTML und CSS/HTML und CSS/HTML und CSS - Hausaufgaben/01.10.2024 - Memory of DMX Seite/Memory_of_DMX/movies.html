Filme

DMX war nicht nur ein einflussreicher Musiker, sondern auch ein talentierter Schauspieler. Er spielte in mehreren erfolgreichen Filmen mit, die seine Vielseitigkeit und Präsenz auf der Leinwand unter Beweis stellten.

Zu seinen bekanntesten Filmen gehören:

  • Romeo Must Die (2000)
  • Cradle to Grave / Born 2 Die (2003)
  • Never Die Alone (2004)
